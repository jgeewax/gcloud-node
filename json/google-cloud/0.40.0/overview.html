First, install google-cloud with npm and require it into your project:

$ npm install --save google-cloud
var gcloud = require('google-cloud');

If you are running your app on Google Compute Engine, you won't need to worry about supplying connection configuration options to gcloud-nodeâ€” we figure that out for you.

However, if you're running your app elsewhere, you will need to provide project details to authenticate API requests.

// Compute Engine var gcloud = require('google-cloud'); // Elsewhere var gcloud = require('google-cloud')({ projectId: 'project-id', keyFilename: '/path/to/keyfile.json' });

The full set of options which can be passed to gcloud-node and sub-modules are outlined here.
